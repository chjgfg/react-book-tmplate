# ✅ `/app/checkout/page.tsx（美化完整版）`

👉 **可直接整体替换**

```tsx
"use client";

import { useCart } from "@/app/providers/CartProvider";
import { useRouter } from "next/navigation";

export default function CheckoutPage() {
  const { items, clearCart } = useCart();
  const router = useRouter();

  const totalPrice = items.reduce(
    (sum, item) => sum + item.price * item.count,
    0
  );

  // 防止直接访问
  if (items.length === 0) {
    return (
      <div className="flex flex-col items-center justify-center py-32 text-gray-500">
        <div className="text-5xl mb-4">💳</div>
        <p className="mb-6">没有需要支付的商品</p>
        <button
          onClick={() => router.push("/")}
          className="text-blue-600 hover:underline"
        >
          返回首页 →
        </button>
      </div>
    );
  }

  const handlePay = () => {
    clearCart();
    router.push("/success");
  };

  return (
    <div className="max-w-3xl mx-auto">
      <h1 className="text-2xl font-bold mb-6">
        💳 确认支付
      </h1>

      <div className="bg-white rounded-xl shadow-sm p-6 space-y-6">
        {/* 商品列表 */}
        <div>
          <h2 className="font-semibold mb-3">
            商品信息
          </h2>

          <div className="border rounded-lg divide-y">
            {items.map((item) => (
              <div
                key={item.id}
                className="flex justify-between py-3 px-4 text-sm"
              >
                <span>
                  {item.title} × {item.count}
                </span>
                <span className="font-medium">
                  ¥ {item.price * item.count}
                </span>
              </div>
            ))}
          </div>
        </div>

        {/* 收货信息 */}
        <div>
          <h2 className="font-semibold mb-3">
            收货信息
          </h2>

          <div className="space-y-4">
            <input
              placeholder="收货人姓名"
              className="w-full border rounded-lg px-4 py-2 focus:outline-none focus:ring-2 focus:ring-blue-500"
            />
            <input
              placeholder="手机号"
              className="w-full border rounded-lg px-4 py-2 focus:outline-none focus:ring-2 focus:ring-blue-500"
            />
            <input
              placeholder="收货地址"
              className="w-full border rounded-lg px-4 py-2 focus:outline-none focus:ring-2 focus:ring-blue-500"
            />
          </div>
        </div>

        {/* 总价 */}
        <div className="flex justify-between items-center border-t pt-4">
          <span className="text-gray-600">
            应付总额
          </span>
          <span className="text-2xl font-bold text-red-500">
            ¥ {totalPrice}
          </span>
        </div>

        {/* 支付按钮 */}
        <button
          onClick={handlePay}
          className="w-full bg-blue-600 text-white py-3 rounded-lg text-lg hover:bg-blue-700 transition"
        >
          确认支付
        </button>
      </div>
    </div>
  );
}
```

# 🎨 Tailwind 样式逐块讲解（非常重要）

下面这些 **你以后 80% 的页面都靠它们了**。

---

## 1️⃣ 页面容器 & 卡片

```ts
max-w-3xl mx-auto
```

* `max-w-3xl`：限制宽度（阅读友好）
* `mx-auto`：水平居中

```ts
bg-white rounded-xl shadow-sm p-6
```

* `bg-white`：卡片背景
* `rounded-xl`：大圆角（现代感）
* `shadow-sm`：轻阴影（层级）
* `p-6`：内边距

---

## 2️⃣ 商品列表分割线

```ts
border rounded-lg divide-y
```

* `border`：整体边框
* `rounded-lg`：小圆角
* `divide-y`：**子项自动加横线**（非常常用）

---

## 3️⃣ 输入框（真实产品级）

```ts
focus:outline-none focus:ring-2 focus:ring-blue-500
```

* 去掉默认 outline
* 获得「聚焦高亮」效果
* **这是企业项目标准写法**

---

## 4️⃣ 总价区域（视觉聚焦）

```ts
border-t pt-4
```

* `border-t`：顶部横线
* `pt-4`：拉开空间

```ts
text-2xl font-bold text-red-500
```

* 字号 + 颜色 → **告诉用户这是最重要的信息**

---

## 5️⃣ 主操作按钮（CTA）

```ts
bg-blue-600 hover:bg-blue-700 transition
```

* 蓝色 = 主操作
* hover 有反馈
* `transition` 让变化自然
